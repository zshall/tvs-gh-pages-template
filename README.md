# Television Simulator GitHub Pages Template

This repository is a starter template that you can use to run [Television Simulator](https://docs.tvs.gcpw.art/) on GitHub Pages.

## Getting started

Create a new repository from this template using **Use this template** > **Create a new repository**.

<img width="589" height="337" alt="image" src="https://github.com/user-attachments/assets/b56a4ad3-1308-4689-8e9d-ac2df96522fa" />

## GitHub Pages configuration

Make sure that GitHub Pages is turned on:

* Go to Settings > Pages
* Under "Branch" choose `main`
* Save.
* You should be able to view the site by clicking "Visit Site"

<img width="1501" height="624" alt="image" src="https://github.com/user-attachments/assets/3d6f5c29-f3f2-4925-951a-903851c59666" />

## Customizing your TVS instance

Edit `config.tvs.yml` and your changes will show up on the web.

An easy way to do this is via the GitHub web editor. You can open this by pressing `.` from the "Code" tab in GitHub.
